# ESTRO Website UI Refresh

## Goal

Redesign the website aesthetically while preserving the existing brand identity.

This is NOT a rebrand.

Keep the existing messaging, content and structure.

The objective is to elevate the visual quality and create a fresher, more innovative and culturally relevant experience consistent with a contemporary creative communication agency.

---

## Important rules

* DO NOT modify text content.
* DO NOT rewrite copy.
* DO NOT change page structure.
* DO NOT add new sections.
* DO NOT modify business logic.
* DO NOT touch SEO.
* Preserve URLs.
* Preserve functionality.

Only improve the visual layer.

---

## Available assets

Use resources already available in the repository:

* `/assets` → logo files
* Visual Identity PDF

Do not use external assets unless absolutely necessary.

---

## Brand DNA

ESTRO transforms ideas into form.

Keywords:

* ideas

The website should feel like a modern creative communication agency rather than a corporate service company.

## Color System

```css
--night: #121212;
--white: #FFFFFF;
--spring-green: #00FF7F;
--electric-indigo: #6300FF;
--raspberry: #DA0759;
```

Use Night as the dominant background.

Accent colors should be used sparingly.

Prefer high contrast.
